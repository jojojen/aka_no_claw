# Reserved Launchers

These scripts are kept only as historical references.

Do not use them for normal OpenClaw operation.

Supported entry points are:

- First setup / cold start: `launchers/start-mac-mini-stack.command`
  or `launchers/start-rpi5-stack.sh`
- Live restart: Telegram `/restartall`, web console restart, or
  `POST http://127.0.0.1:8781/api/command/restartall`
